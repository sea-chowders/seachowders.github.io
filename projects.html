Shopping Cart

This is a simple shoppig cart web application that I developed for my ASP.NET class. It reads from a SQL Server database, and populates the fields in real time. Users are able to view products, add/remove items from the cart, and view information pertaining to the items in their cart. The project wasn't graded on design at all, and unfortunately I did not have time to make the project look good. In the future I would love to clean up the design of the site.
